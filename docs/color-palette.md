# Brand Color Palette

## CSS Custom Properties

```css
:root {
  --brand-logo: #1D4ED8; /* Logo, primary CTAs, headings, key UI elements, links. */
  --brand-highlighting-key-insights: #FBBF24; /* Highlighting key insights, callouts, accents in illustrations, secondary buttons. */
  --brand-body-text: #1F2937; /* Body text, paragraphs, secondary information. */
  --brand-main-background-color-for-the-application-and-website: #F9FAFB; /* Main background color for the application and website. */
}
```

## Palette Reference

| Hex | Usage |
|-----|-------|
| `#1D4ED8` | Logo, primary CTAs, headings, key UI elements, links. |
| `#FBBF24` | Highlighting key insights, callouts, accents in illustrations, secondary buttons. |
| `#1F2937` | Body text, paragraphs, secondary information. |
| `#F9FAFB` | Main background color for the application and website. |

## Imagery Direction
Imagery should be clean, bright, and focused on the user's feeling of empowerment. Use photos of diverse freelancers and small business owners looking confident and in control, not stressed. Avoid legal clichés like gavels, scales, or dusty books. Utilize abstract, geometric illustrations to visualize concepts like risk analysis (e.g., highlighting a path through a maze) or clarity (e.g., untangling lines).
